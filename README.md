# yolo-app
YOLOv8n object detection web app deployed using Flask and Docker, with a simple user interface for image based detection.

# YOLOv8 Object Detection Web App

This project presents a lightweight web application for object detection using a custom-trained YOLOv8 model. The system allows users to upload an image through a simple web interface and receive detection results with bounding boxes.


## Model Description

The application uses a YOLOv8-based deep learning model trained for object detection tasks. The model processes input images and identifies objects with high accuracy, returning annotated results.

##  Features

- Image upload via web interface  
- Real-time object detection  
- Visualization of results with bounding boxes  
- Dockerized deployment for portability
